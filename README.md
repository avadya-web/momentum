# Momentum

Momentum is a personal planning PWA for work streams, critical tasks, recurring rhythms, personal time, and daily timeboxing.

## Use

Open the published GitHub Pages URL in a browser. On iPhone, use Safari and choose Share > Add to Home Screen. On Android, use Chrome and choose Install app or Add to Home screen.

This version supports Supabase login and per-user sync. Run `supabase-schema.sql` in the Supabase SQL editor before using sync.
